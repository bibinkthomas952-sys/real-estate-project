# Real Estate Management System

This is a Django-based Real Estate Management System where:

- Admin can add properties
- Users can view properties
- Admin dashboard is available
- Property list, contact page, buy page included

## Technologies Used
- Python
- Django
- HTML, CSS
- SQLite

## How to Run

1. Install Python
2. Install Django:
   pip install django
3. Run:
   python manage.py runserver
4. Open browser:
   http://127.0.0.1:8000/
